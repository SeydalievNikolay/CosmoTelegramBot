# Telegram-bot для кабинета косметолога"

## Функционал:
* Запуск бота
* Возможность выбора даты и времени
* Выбор процедуры
* Отмена или перенос запси
* Раздел помощи

## Используемые технологии и библиотеки:

- Java 17 и выше
- Telegram API
- Spring Boot
- Lombok
- DB PostgreSQL
- Docker
- Maven
- Hibernate
