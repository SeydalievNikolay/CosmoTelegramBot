# Telegram-bot для кабинета косметолога"

## Функционал:
* Запуск бота
* Возможность выбора даты и времени
* Выбор процедуры
* Раздел помощи

## Используемые технологии и библиотеки:

- Java 21 и выше
- Spring Boot
- Lombok
- DB PostgreSQL
- Docker
- Maven
- Hibernate
